设计模式——工厂模式

发表于 2018-04-04 | 分类于 设计模式

工厂模式（Factory Pattern）是 Java 中最常用的设计模式之一。属于创建型模式，它提供了一种创建对象的最佳方式。在工厂模式中，我们在创建对象时不会对客户端暴露创建逻辑，并且是通过使用一个共同的接口来指向新创建的对象。工厂模式根据抽象程度的不同分为三种：简单工厂模式、静态工厂方法模式、抽象工厂模式。
